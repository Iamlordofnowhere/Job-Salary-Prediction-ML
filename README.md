# 💼 Предсказание зарплат специалистов

## Описание
ML модель для предсказания зарплат специалистов на основе характеристик работы.
Датасет содержит 250,000 записей с признаками: должность, опыт, образование, локация и др.

## Цель
Предсказать зарплату с помощью RandomForest + подбор гиперпараметров через Optuna + K-Fold кросс-валидация.

## Инструменты
- Python, pandas, numpy
- matplotlib, scikit-learn, optuna

## Результаты модели

| Метрика | Baseline | Optuna |
|---------|----------|--------|
| R²      | 0.9691   | 0.9494 |
| MAE     | 5,184$   | 6,645$ |
| RMSE    | 6,557$   | 8,386$ |

## Ключевые инсайты
- **Локация** — главный фактор зарплаты (важность 0.31)
- **Опыт** — второй по важности признак (важность 0.21)
- **Размер компании** важнее должности (0.17 vs 0.15)
- **Сертификаты и удалёнка** почти не влияют (< 0.02)
- Модель стабильна: разброс K-Fold ±0.0003

## Данные
[Job Salary Prediction Dataset — Kaggle](https://www.kaggle.com/datasets/aleesha-nadeem/job-salary-prediction-dataset)
